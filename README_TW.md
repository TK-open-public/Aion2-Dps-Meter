# Aion2-Dps-Meter 台服使用指南

## 台服支持说明

Aion2-Dps-Meter 现在支持台服版本的 AION2。以下是使用指南：

## 如何切换到台服模式

1. 运行 Aion2-Dps-Meter
2. 点击界面顶部的地球图标（服务器设置按钮）
3. 在弹出的服务器设置面板中，选择"台服"
4. 点击"保存"按钮
5. 重启 Aion2-Dps-Meter 以应用设置

## 台服特殊说明

1. **数据文件**：台服使用与韩服不同的数据文件，包括技能、怪物和 buff 数据
2. **兼容性**：台服数据正在持续完善中，部分功能可能与韩服有所不同
3. **问题反馈**：如果在台服使用过程中遇到问题，请在 GitHub Issues 中反馈

## 安装步骤

1. 安装 npcap（https://npcap.com/#download），必须勾选 "Install Npcap in WinPcap API-compatible Mode"
2. 下载并安装 Aion2-Dps-Meter
3. 以管理员权限运行程序
4. 切换到台服模式（如上述步骤）
5. 启动 AION2 游戏，进入角色选择界面
6. 进入游戏世界，开始战斗
7. 程序会自动捕获数据包并计算 DPS

## 常见问题

### 1. 切换到台服模式后没有数据显示
- **原因**：可能是台服数据文件未正确加载
- **解决方案**：
  - 确保已重启程序
  - 检查游戏是否正常运行
  - 确保以管理员权限运行程序

### 2. 技能名称显示为数字
- **原因**：台服技能数据可能尚未完全更新
- **解决方案**：这些数字是技能的 ID，随着数据的完善会逐渐显示为正确的技能名称

### 3. 怪物名称显示不正确
- **原因**：台服怪物数据可能尚未完全更新
- **解决方案**：随着数据的完善，怪物名称会逐渐显示正确

## 数据更新

台服数据会定期更新，以支持最新的游戏内容。如果您发现数据需要更新，请在 GitHub Issues 中反馈。

## 注意事项

1. 程序需要管理员权限才能正常捕获网络数据包
2. 切换服务器类型后必须重启程序才能生效
3. 台服数据正在持续完善中，部分功能可能与韩服有所不同
4. 使用本程序时请遵守游戏的使用条款
